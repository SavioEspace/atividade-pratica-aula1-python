# 🚀 Aula 1 — Análise Exploratória

## 📌 Objetivo
Realizar uma análise inicial do dataset, entendendo sua estrutura e extraindo insights básicos.

## 🔍 Atividades realizadas
- Leitura e inspeção dos dados
- Identificação de colunas e tipos
- Verificação de valores nulos
- Cálculo de KPIs (receita, lucro, ticket médio)
- Análise por canal, produto e UF
- Primeiras interpretações de negócio

## 🧠 Principais aprendizados
- Importância do entendimento inicial dos dados
- Como identificar inconsistências rapidamente
- Uso de Pandas para análise exploratória

---

# 📊 Aula 2 — Preparação e Análise de Dados

## 📌 Objetivo
Preparar os dados para análise e gerar insights com base em filtros, rankings e visualizações.

## 🔍 Atividades realizadas
- Check-up completo do dataset
- Conversão de dados (texto → numérico)
- Tratamento de inconsistências
- Criação de filtros (RJ, RJ Online)
- Geração de rankings (Top N)
- Análise temporal
- Interpretação gerencial dos resultados

## 🧠 Principais aprendizados
- Dados inconsistentes podem distorcer análises
- Importância da padronização de categorias
- Como transformar dados em decisão

---

# 🧹 Aula 3 — Limpeza e Qualidade dos Dados

## 📌 Objetivo
Realizar a limpeza completa de uma base de dados bruta, garantindo qualidade para análise.

## 🔍 Atividades realizadas
- Diagnóstico inicial da base
- Conversão de tipos (datas e valores monetários)
- Tratamento de valores ausentes
- Remoção de duplicidades
- Padronização de categorias (UF, canal, categoria)
- Criação de variáveis derivadas (ano, mês, margem)
- Validação final da base
- Exportação da base limpa

## 📁 Arquivo gerado
- `vendas_brasil_aula3_clean.csv`

## 🧠 Principais aprendizados
- Limpeza de dados é a base de qualquer análise confiável
- Pequenos erros podem gerar grandes distorções
- A preparação dos dados é a etapa mais crítica do processo analítico

---

# 🏁 Conclusão

As atividades demonstram a evolução do processo analítico:

1. **Explorar os dados** (Aula 1)  
2. **Preparar e analisar** (Aula 2)  
3. **Garantir qualidade e confiabilidade** (Aula 3)  

A qualidade da análise depende diretamente da qualidade dos dados, e todo o processo reforça a importância da limpeza como **fundação invisível** da análise.
